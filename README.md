## Featured Project

### [QAL — Memristor-Constrained CNN on CIFAR-10](https://github.com/Qaiser-baitham/QAL)

> Simulating a **Pt/ZnO/MoO₃/Pt** bilayer memristor device and using its real conductance levels (LTP/LTD) to constrain FC layer weights in a CNN trained on CIFAR-10.

- Real device data (Excel) → conductance extraction → weight quantization levels
- Straight-Through Estimator (STE) for training through discrete weights
- Publication-quality plots + Origin-compatible Excel exports
- **Author: QAISER ABBAS**
